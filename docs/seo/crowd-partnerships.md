# Крауд и партнёрские ссылки (фаза E3)

Цель: внешние упоминания ведут на **глубокие** страницы (лендинги услуг `/uslugi/.../`, каталог с фильтром), а не только на главную.

## Типы площадок (примерный список)

- Локальные СМИ и городские порталы (рубрика «авто», партнёрские материалы).
- Автоклубы, форумы (по правилам площадки, без спама).
- Партнёры (заправки, магазины запчастей) — блок «где обслужиться» со ссылкой на каталог или услугу.

## Формат ссылки

- Лендинг: `https://<домен>/uslugi/<slug>/` (при мультигороде при необходимости `?city=...`).
- Каталог с услугой: `https://<домен>/sto/?service=<slug>` (слаг категории) или `?cat=<id>` для числового id из админки; не путать слаг с `cat`.
- Якорный текст: естественный (название услуги + город), без переспама одной и той же фразы.

## Трекинг (вне репозитория)

Ведите таблицу: площадка, дата, URL целевой страницы, тип размещения, ответственный. UTM-метки — по единому шаблону согласованности с аналитикой, чтобы не ломать canonical (не дублировать индексируемые URL с разными UTM без политики).

## Риски

- Покупка «сотен ссылок» — не использовать.
- Одинаковый текст на десятках сайтов — избегать; лучше уникальные короткие абзацы.
