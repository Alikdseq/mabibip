🚗 ДВИГАТЕЛЬ И ЕГО РАБОТА (Моторный цех)
Старт и холостой ход:

троит двигатель → Диагностика двигателя / Замена свечей / Замена катушек

двигатель троит на холодную → Замена форсунок / Замена свечей накала (дизель)

двигатель троит на горячую → Замена катушки зажигания / Замена форсунки

плавают обороты → Чистка дроссельной заслонки / Чистка клапана холостого хода

глохнет на холостых → Чистка дроссельной заслонки / Замена ДМРВ

глохнет при нажатии на газ → Замена топливного насоса / Замена ДПДЗ

не заводится → Диагностика электрооборудования / Замена стартера

стартер крутит но не схватывает → Диагностика топливной системы / Замена бензонасоса

не заводится на горячую → Замена датчика коленвала / Ремонт стартера

схватывает и глохнет → Обходчик иммобилайзера / Диагностика сигнализации

долго крутит стартер перед запуском → Замена обратного клапана бензонасоса

Звуки:

стук в двигателе → Дефектовка ДВС / Замена гидрокомпенсаторов

стук на холодную → Замена гидрокомпенсаторов / Замена поршневой (диагностика)

стук на горячую → Замена вкладышей / Замена поршневой

цокот в двигателе → Замена форсунок / Регулировка клапанов

свист из под капота → Замена ремня генератора / Замена натяжного ролика

стрекот в двигателе → Замена цепи ГРМ / Замена фазорегулятора

Дым и запах:

дымит двигатель → Диагностика двигателя

белый дым из выхлопной → Замена прокладки ГБЦ / Диагностика ЕГР

сизый дым из выхлопной → Замена маслосъемных колпачков / Раскоксовка

черный дым из выхлопной → Замена форсунок / Замена турбины / Чистка ЕГР

запах бензина в салоне → Ремонт топливной магистрали / Замена адсорбера

запах выхлопных газов в салоне → Ремонт системы выхлопа (прогар прокладки)

запах горелого масла в салоне → Замена прокладки клапанной крышки / Устранение течи масла

Расход и динамика:

жрет масло → Замена МСК / Замена маслосъемных колец (капиталка)

повышенный расход масла → Дефектовка турбины / Замена сальников

большой расход топлива → Замена лямбда-зонда / Чистка форсунок

потеря мощности → Замена катализатора / Диагностика наддува

не тянет двигатель → Замена бензонасоса / Замена ДМРВ

провалы при разгоне → Замена свечей / Замена катушек / Замена топливного фильтра

детонация → Заливка качественного топлива / Замена датчика детонации

🛞 ХОДОВАЯ ЧАСТЬ (Подвеска и рулевое)
Звуки снизу:

стук в подвеске → Диагностика ходовой части

стук спереди → Замена стоек стабилизатора / Замена шаровых опор

стук сзади → Замена втулок стабилизатора / Замена сайлентблоков балки

стук на кочках → Замена стоек стабилизатора / Замена амортизаторов

скрип в подвеске → Замена сайлентблоков / Замена шаровых опор

гул при движении → Замена ступичного подшипника

гул на скорости → Замена ступицы / Замена редуктора

вой заднего моста → Ремонт редуктора / Замена масла в редукторе

хруст при повороте → Замена ШРУСа / Замена гранаты

щелчки при повороте руля → Замена ШРУСа

стук в рулевой рейке → Ремонт рулевой рейки / Замена рулевых наконечников

Управляемость:

вибрация на скорости → Балансировка колес / Замена ШРУСа

бьет руль → Балансировка колес / Правка дисков

уводит машину в сторону → Сход развал / Замена сайлентблоков

жрет резину → Сход развал 3D / Ремонт подвески

тугой руль → Ремонт рулевой рейки / Замена насоса ГУР / Замена крестовины кардана

люфт в рулевом управлении → Замена рулевых тяг / Ремонт рулевой рейки

не возвращается руль → Ремонт рулевой рейки / Сход развал

🛑 ТОРМОЗНАЯ СИСТЕМА
Звуки:

свист тормозов → Замена тормозных колодок / Чистка тормозов

скрип при торможении → Замена тормозных колодок / Замена тормозных дисков

скрежет при торможении → Срочная замена колодок и дисков

стук при торможении → Переборка суппорта / Замена направляющих суппорта

Поведение педали:

мягкая педаль тормоза → Прокачка тормозов / Замена тормозной жидкости

педаль тормоза проваливается → Замена главного тормозного цилиндра (ГТЦ) / Поиск утечки тормозной жидкости

тугая педаль тормоза → Замена вакуумного усилителя тормозов (ВУТ)

вибрация при торможении → Замена тормозных дисков / Проточка тормозных дисков

биение в педаль тормоза → Замена тормозных дисков

Прочее:

машину уводит при торможении → Переборка суппорта / Замена тормозного шланга

горит лампа abs → Диагностика ABS / Замена датчика ABS

горит ручник → Проверка уровня тормозной жидкости / Замена колодок с датчиком износа

не работает ручник → Регулировка ручника / Замена троса ручника

⚙️ ТРАНСМИССИЯ (Коробка передач)
АКПП / DSG / Вариатор:

пинается коробка → Замена масла в АКПП / Ремонт гидроблока

толчки при переключении → Замена масла в АКПП / Адаптация АКПП

буксует коробка автомат → Замена фрикционов АКПП (Капремонт АКПП)

не включается передача → Диагностика АКПП / Замена соленоидов

пропала задняя скорость → Капитальный ремонт АКПП

аварийный режим акпп → Компьютерная диагностика АКПП

вариатор буксует → Замена ремня вариатора

вариатор воет → Замена подшипников вариатора

dsg дергается → Замена сцепления DSG / Ремонт мехатроника

МКПП:

скрежет при включении передачи → Замена сцепления / Ремонт МКПП (синхронизаторы)

туго включаются передачи → Замена троса сцепления / Ремонт кулисы

пробуксовка сцепления → Замена сцепления (комплект)

ведет сцепление → Прокачка сцепления / Замена главного цилиндра сцепления

вой мкпп → Замена масла в МКПП / Замена подшипников МКПП

выбивает передачу → Ремонт кулисы / Ремонт МКПП (износ вилок)
